## 全体フロー シーケンス図

```mermaid
sequenceDiagram
    autonumber
    participant Unity
    participant Presenter as BattleScenePresenter
    participant Repo as BattleSequenceRepository
    participant StateMachine as BattlePhaseStateMachine
    participant Sequence as BattleSequenceModel
    participant Phase as BattlePhaseModelBase<br/>(TimeLimitBattlePhaseModel)
    participant BindingMap as BattleTimelineBindingMap
    participant Director as PlayableDirector
    participant ScenarioScene as ScenarioScenePresenter

    Note over Unity,ScenarioScene: ゲーム開始
    Unity->>Presenter: Start() / InitializeAndStart()
    
    rect rgb(230, 245, 255)
        Note over Presenter,Repo: ステージ解決 & シーケンスロード
        Presenter->>Presenter: ResolveStageModel()
        Presenter->>Repo: Load(waySequenceAddress)
        Repo-->>Presenter: waySequence (BattleSequenceModel)
        Presenter->>Repo: Load(bossSequenceAddress)
        Repo-->>Presenter: bossSequence (BattleSequenceModel)
    end

    rect rgb(255, 245, 230)
        Note over Presenter,Director: 道中シーケンス再生
        Presenter->>StateMachine: PlaySequence(waySequence)
        StateMachine->>Sequence: Reset()
        
        loop 各フェーズを順番に処理
            StateMachine->>Sequence: MoveNext()
            Sequence-->>StateMachine: phase (BattlePhaseModelBase)
            StateMachine->>Phase: ResolveTimeline()
            Phase-->>StateMachine: TimelineAsset
            StateMachine->>Director: playableAsset = timeline
            StateMachine->>BindingMap: ApplyBindings(director, timeline)
            BindingMap->>Director: SetGenericBinding()
            StateMachine->>Director: Play()
            StateMachine->>Phase: Enter(director)
            Phase->>Phase: OnEnter() [タイマー開始]
            
            Note over Phase: 制限時間経過
            Phase-->>StateMachine: OnExitPhase (CompletePhase)
            StateMachine->>Phase: Exit()
        end
        
        StateMachine->>Director: Stop()
        StateMachine-->>Presenter: OnSequenceCompleted(Way)
    end

    rect rgb(245, 255, 230)
        Note over Presenter,ScenarioScene: 道中→ボス間シナリオ遷移
        Presenter->>Presenter: TransitionToScenario(callback)
        Presenter->>Unity: SceneManager.LoadScene(Scenario, Additive)
        Unity-->>Presenter: OnScenarioSceneLoaded
        Presenter->>ScenarioScene: OnScenarioCompleted.Subscribe()
        
        Note over ScenarioScene: シナリオ再生
        ScenarioScene-->>Presenter: OnScenarioCompleted
        Presenter->>Presenter: pendingCallback.Invoke()
    end

    rect rgb(255, 230, 230)
        Note over Presenter,Director: ボスシーケンス再生
        Presenter->>StateMachine: PlaySequence(bossSequence)
        StateMachine->>Sequence: Reset()
        
        loop 各フェーズを順番に処理
            StateMachine->>Sequence: MoveNext()
            Sequence-->>StateMachine: phase
            StateMachine->>Phase: ResolveTimeline()
            StateMachine->>Director: playableAsset = timeline
            StateMachine->>BindingMap: ApplyBindings()
            StateMachine->>Director: Play()
            StateMachine->>Phase: Enter(director)
            
            Note over Phase: 制限時間経過
            Phase-->>StateMachine: OnExitPhase
            StateMachine->>Phase: Exit()
        end
        
        StateMachine->>Director: Stop()
        StateMachine-->>Presenter: OnSequenceCompleted(Boss)
    end

    rect rgb(230, 255, 245)
        Note over Presenter,ScenarioScene: ボス→次ステージ シナリオ遷移
        Presenter->>Presenter: TransitionToScenario(CompleteStage)
        Presenter->>Unity: SceneManager.LoadScene(Scenario, Additive)
        Unity-->>Presenter: OnScenarioSceneLoaded
        ScenarioScene-->>Presenter: OnScenarioCompleted
    end

    rect rgb(240, 240, 255)
        Note over Presenter: バトル完了処理
        Presenter->>Presenter: CompleteStage()
        Presenter->>Presenter: stageModel.Clear()
        alt 次ステージあり
            Presenter->>Presenter: ResolveStageModel()
            Presenter->>StateMachine: PlaySequence(waySequence)
        else 最終ステージクリア
            Presenter-->>Unity: OnBattleCompleted
        end
    end
```

## フェーズ内 Timeline再生 詳細

```mermaid
sequenceDiagram
    autonumber
    participant StateMachine as BattlePhaseStateMachine
    participant Phase as BattlePhaseModelBase
    participant Definition as BattlePhaseDefinition
    participant Builder as BattleTimelineBuilderAsset
    participant BindingMap as BattleTimelineBindingMap
    participant Director as PlayableDirector

    StateMachine->>Phase: ResolveTimeline()
    
    Phase->>Definition: CreateTimeline()
    Definition->>Builder: BuildTimeline()
    Note over Builder: 6種類のTrackを動的生成<br/>Signal/Animation/Activation/<br/>Audio/Control/EnemySpawn
    Builder-->>Definition: TimelineAsset (runtime)
    Definition-->>Phase: TimelineAsset
    
    Phase-->>StateMachine: resolvedTimeline
    
    StateMachine->>Director: playableAsset = resolvedTimeline
    StateMachine->>Director: time = 0
    StateMachine->>Director: Evaluate()
    StateMachine->>Director: extrapolationMode = Loop
    
    StateMachine->>BindingMap: ApplyBindings(director, timeline)
    
    loop timeline.outputs を列挙
        BindingMap->>BindingMap: TryGetBinding(trackName)
        BindingMap->>Director: SetGenericBinding(track, target)
    end
    
    StateMachine->>Director: Play()
```

## Signal処理フロー

```mermaid
sequenceDiagram
    autonumber
    participant Director as PlayableDirector
    participant Emitter as SignalEmitter
    participant Receiver as Stage1SignalReceiver
    participant Handler as ゲームロジック

    Director->>Emitter: Timeline再生中にマーカー到達
    Emitter->>Receiver: OnNotify(origin, notification, context)
    
    alt EnemySpawnSignal の場合
        Receiver->>Receiver: HandleEnemySpawn(signal)
        Note over Receiver: signal.SpawnPosition<br/>signal.MaxHp を取得
        Receiver->>Handler: 敵生成処理
    else 汎用SignalAsset の場合
        Receiver->>Receiver: HandleSignal(emitter)
        Note over Receiver: emitter.asset.name で分岐
        alt name = "Stage1_Sample"
            Receiver->>Receiver: SampleFunc()
        else その他
            Receiver->>Receiver: LogWarning (未定義)
        end
    end
```

## クラス構成図

```mermaid
classDiagram
    direction TB
    
    class BattleScenePresenter {
        -BattlePhaseStateMachine phaseStateMachine
        -BattleSequenceRepository sequenceRepository
        -StageModel stageModel
        -BattleSequenceModel waySequence
        -BattleSequenceModel bossSequence
        +IObservable~Unit~ OnBattleCompleted
        +InitializeAndStart()
        +TransitionToScenario(callback)
    }
    
    class BattlePhaseStateMachine {
        -PlayableDirector playableDirector
        -BattleTimelineBindingMap bindingMap
        -BattleSequenceModel activeSequence
        -BattlePhaseModelBase activePhase
        +IObservable~BattlePhaseModelBase~ OnPhaseStarted
        +IObservable~BattleSequenceType~ OnSequenceCompleted
        +PlaySequence(sequence)
        +Stop()
    }
    
    class BattleSequenceModel {
        -IReadOnlyList~BattlePhaseModelBase~ phases
        -int currentIndex
        +BattleSequenceType SequenceType
        +MoveNext() BattlePhaseModelBase
        +Reset()
    }
    
    class BattlePhaseModelBase {
        <<abstract>>
        #BattlePhaseDefinition Definition
        #PlayableDirector Director
        +string PhaseId
        +IObservable~Unit~ OnExitPhase
        +ResolveTimeline() TimelineAsset
        +Enter(director)
        +Exit()
        +Dispose()
        #OnEnter()*
        #CompletePhase()
    }
    
    class TimeLimitBattlePhaseModel {
        #OnEnter()
    }
    
    class BattleTimelineBindingMap {
        -List~BindingEntry~ bindings
        +ApplyBindings(director, timeline)
    }
    
    class BattleSequenceAsset {
        <<ScriptableObject>>
        +BattleSequenceType SequenceType
        +IReadOnlyList~BattlePhaseDefinition~ Phases
    }
    
    class BattlePhaseDefinition {
        +string PhaseId
        +float TimeLimitSeconds
        +BattleTimelineBuilderAsset TimelineBuilder
        +CreateTimeline() TimelineAsset
    }
    
    class BattleTimelineBuilderAsset {
        <<ScriptableObject>>
        -List~SignalTrackDefinition~ signalTracks
        -List~AnimationTrackDefinition~ animationTracks
        -List~ActivationTrackDefinition~ activationTracks
        -List~AudioTrackDefinition~ audioTracks
        -List~ControlTrackDefinition~ controlTracks
        -List~EnemySpawnTrackDefinition~ enemySpawnTracks
        +BuildTimeline() TimelineAsset
    }

    class Stage1SignalReceiver {
        <<MonoBehaviour>>
        +OnNotify(origin, notification, context)
        -HandleSignal(emitter)
        -HandleEnemySpawn(signal)
    }

    class EnemySpawnSignal {
        <<ScriptableObject>>
        +Vector3 SpawnPosition
        +int MaxHp
        +SetProperties(position, hp)
    }
    
    BattleScenePresenter --> BattlePhaseStateMachine : uses
    BattleScenePresenter --> BattleSequenceModel : manages
    BattlePhaseStateMachine --> BattleSequenceModel : plays
    BattlePhaseStateMachine --> BattlePhaseModelBase : controls
    BattlePhaseStateMachine --> BattleTimelineBindingMap : uses
    BattleSequenceModel --> BattlePhaseModelBase : contains
    BattlePhaseModelBase <|-- TimeLimitBattlePhaseModel : extends
    BattlePhaseModelBase --> BattlePhaseDefinition : uses
    BattlePhaseDefinition --> BattleTimelineBuilderAsset : uses
    BattleSequenceAsset --> BattlePhaseDefinition : contains
    BattleTimelineBuilderAsset ..> EnemySpawnSignal : creates
    Stage1SignalReceiver ..> EnemySpawnSignal : handles
```

## 状態遷移図

```mermaid
stateDiagram-v2
    [*] --> Idle: シーン開始
    
    state "道中フェーズ" as WayPhases {
        [*] --> WayPhase1: PlaySequence(waySequence)
        WayPhase1 --> WayPhase2: OnExitPhase (時間切れ)
        WayPhase2 --> WayPhaseN: OnExitPhase
        WayPhaseN --> [*]: MoveNext() = null
    }
    
    Idle --> WayPhases: InitializeAndStart()
    WayPhases --> ScenarioWayToBoss: OnSequenceCompleted(Way)
    
    state "シナリオ遷移 (道中→ボス)" as ScenarioWayToBoss {
        [*] --> LoadingScenario1: SceneManager.LoadScene(Additive)
        LoadingScenario1 --> WaitingScenario1: OnScenarioSceneLoaded
        WaitingScenario1 --> [*]: OnScenarioCompleted
    }
    
    state "ボスフェーズ" as BossPhases {
        [*] --> BossPhase1: PlaySequence(bossSequence)
        BossPhase1 --> BossPhase2: OnExitPhase
        BossPhase2 --> BossPhaseN: OnExitPhase
        BossPhaseN --> [*]: MoveNext() = null
    }
    
    ScenarioWayToBoss --> BossPhases: StartBossSequence()
    BossPhases --> ScenarioBossToNext: OnSequenceCompleted(Boss)
    
    state "シナリオ遷移 (ボス→次)" as ScenarioBossToNext {
        [*] --> LoadingScenario2: SceneManager.LoadScene(Additive)
        LoadingScenario2 --> WaitingScenario2: OnScenarioSceneLoaded
        WaitingScenario2 --> [*]: OnScenarioCompleted
    }
    
    state "ステージ完了判定" as CompleteCheck {
        [*] --> CheckNext: stageModel.Clear()
        CheckNext --> NextStage: 次ステージあり
        CheckNext --> AllCleared: 最終ステージ
    }
    
    ScenarioBossToNext --> CompleteCheck: CompleteStage()
    CompleteCheck --> WayPhases: 次ステージ開始
    CompleteCheck --> Completed: 全クリア
    Completed --> [*]: OnBattleCompleted
```
